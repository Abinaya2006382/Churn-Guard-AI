# ChurnGuard AI
## Multi-Agent Customer Retention Assistant


## Problem Statement

Businesses lose customers because they cannot identify customers who are likely to leave.

ChurnGuard AI helps companies predict customer churn and provides retention strategies using AI agents.


## Objective

- Predict customer churn risk
- Analyze customer behavior
- Provide personalized retention recommendations
- Support business decisions


## Project Features

✅ Machine Learning Churn Prediction  
✅ Multi-Agent AI System  
✅ MCP Customer Data Server  
✅ Agent Skills  
✅ Automated Retention Recommendations  


## Architecture

User

↓

ChurnGuard AI

↓

Data Agent → Prediction Agent → Recommendation Agent

↓

Retention Strategy


## Agents

### Data Agent
Analyzes customer information and prepares data.

### Prediction Agent
Predicts churn probability using Random Forest model.

### Recommendation Agent
Suggests actions to retain customers.


## Machine Learning Model

Algorithm:

Random Forest Classifier


Results:

Training Accuracy: 93.37%

Testing Accuracy: 93.21%


## Agent Skills

Functions:

- analyze_customer()
- predict_churn_risk()
- recommend_strategy()


## MCP Server

Customer Data MCP Server provides customer details securely to agents.
